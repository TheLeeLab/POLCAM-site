Software

Three software packages are available for POLCAM image processing:

  • POLCAM-SR: A MATLAB plugin for single-molecule and diffraction-limited polarisation camera image processing (GitHub ).
  • POLCAM-Live: A standalone MATLAB application for live polarisation camera image processing and visualization (GitHub).
  • napari-polcam: A napari plugin for fast processing of multi-dimensional polarisation camera image data (GitHub).

POLCAM-SR

POLCAM-SR is a MATLAB plugin for single-molecule and diffraction-limited polarisation camera image processing.

The plugin installer and source code are available on the POLCAM-SR repository on GitHub.

POLCAM-Live

POLCAM-Live is a standalone MATLAB application for live polarisation camera image processing and visualization during experiments.

The installer and source code are available on the POLCAM-Live repository on GitHub.

napari-polcam

napari-polcam is a napari plugin for fast processing of multi-dimensional polarisation camera image data.
